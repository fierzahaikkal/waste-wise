# Waste Wise

## Run Locally

Clone the project

```bash
  git clone https://github.com/fierzahaikkal/waste-wise.git
```

Go to the project directory

```bash
  cd waste-wise
```

Install dependencies

```bash
  npm install
```

if not working

```bash
  npm install --legacy-peer-deps
```

Start the server

```bash
  npm run dev
```
